# 🍔 Food Recommender System (Beginner AI Project)

## 📌 Overview

This is a simple **Food Recommendation System** built using Python.
It suggests food items based on user preferences like **taste, cuisine, and budget**.

This project is part of my Artificial Intelligence learning journey and focuses on **basic recommendation logic (no ML yet)**.

---

## 🎯 Features

* Takes user input (taste, cuisine, budget)
* Matches preferences with dataset
* Displays personalized food recommendations
* Beginner-friendly logic using Python

---

## 🧠 How It Works

The system uses:

* Lists & dictionaries (dataset)
* Conditional statements (`if-else`)
* Loops for matching preferences

It compares user input with food data and recommends matching items.
